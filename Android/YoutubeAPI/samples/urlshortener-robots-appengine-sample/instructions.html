Browse Source Code

Checkout Instructions

Prerequisites: install Java 6, Google App Engine, Mercurial, and Maven. You may need to set your JAVA_HOME . 

cd [someDirectory]
hg clone https://code.google.com/p/google-api-java-client.samples/ google-api-java-client-samples
cd google-api-java-client-samples/urlshortener-robots-appengine-sample
mvn clean package

Setup Project in Eclipse 3.5/3.6

Prerequisites: install Eclipse, Google Plugin for Eclipse, and the Mercurial plugin.

  • Setup Eclipse Preferences
    • Window > Preferences... (or on Mac, Eclipse > Preferences...)
    • Select Maven
      • check on "Download Artifact Sources"
      • check on "Download Artifact JavaDoc"
  • Import urlshortener-robots-appengine-sample project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find [someDirectory]/google-api-java-client-samples/urlshortener-robots-appengine-sample and click "Next"
    • Click "Finish"
    • NOTE: please ignore the 4 "The App Engine SDK JAR * is missing in the WEB-INF/lib directory" error messages.
  • Run
    • This application must be deployed to App Engine to work properly!
